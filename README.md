# product_db
# Product Database Management System

## Project Overview

This project is a Product Database Management System developed using MySQL. It is designed to manage products, categories, customers, orders, payments, suppliers, inventory, and returns efficiently.

## Database Name

product_db

## Tables Used

* branches
* categories
* customers
* employee_log
* employees
* inventory
* order_details
* orders
* payments
* products
* returns
* suppliers

## SQL Concepts Used

* DDL Commands (CREATE, ALTER, DROP)
* DML Commands (INSERT, UPDATE, DELETE)
* Joins
* Subqueries
* Correlated Subqueries
* Common Table Expressions (CTE)
* Window Functions
* Triggers
* Aggregate Functions
* Views

## Features

* Product Management
* Customer Management
* Order Tracking
* Payment Records
* Inventory Management
* Supplier Management
* Employee Records
* Return Management
